# ToDo-List-App-Toy-Project
> Vanilla JS와 날씨 API를 활용한 To-Do 리스트 웹 앱입니다. <br/>
> 출처: 노마드코더 강의 활용

<br/>

로컬스토리지에 사용자의 이름과 To-Do 리스트를 저장하여 사용하는 크롬 어플리케이션입니다.<br/>
weather API를 활용하여 사용자의 위치를 저장, 현재 날씨와 온도를 실시간으로 체크합니다.
